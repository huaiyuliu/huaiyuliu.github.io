Contest Details

Individual Design Contest:

Requirement: Student is expected to design a low power, high speed, reconfigurable neuromorphic building block circuit such as 1-to-16-bit multiplication-and-addition unit, 1-to-16-bit activation function, 1-to-16-bit convolution processing element etc. The student can implement the circuit either using analogue or digital circuits. Student is only allowed to use FreePDK45 technology.

The judging criteria is based on the following considerations:

Preliminary Round (submission criteria):

  • Analogue/Mixed-Signal Implementation: Open access database of the schematic and layout

  • Digital Implementation: Verilog code and testbench

  • Details descriptions of the report,

    1.Describe the functionality of the circuit

    2.Describe the design trade-off of the circuit

    Analog Implementation:

    1.Provide important Labels in the schematic

    2.Provide verification results including explanation. The verification includes schematic and post-layout functional verification and variability (includes PVT and Monte Carlo) analysis.

    3.Provide a detail circuit analysis using a mathematic model or with the use of circuit theory

    Digital Implementation:

    1.Annotated Verilog code with explanation in the report.

    2.Digital Synthesis with the FreePDK45nm and report the power, area, speed matrix

    3.Timing and Corner simulations

Final Round (submission and Presentation criteria):

  • Video and Poster demonstration

Group Contest:

Requirement: Team is expected to design system implementation of neuromorphic system for voice activity detection. Team can decide to implement the circuit either using analogue/mixed-signal or digital circuit technique. For analogue/mixed-siginal implementation, team is only allowed to use FreePDK45 technology. For digital implementation, team must demonstrate it in the given FPGA (PYNQ-Z1). For benchmark comparison, students can refer to the open-source dataset [1].

The judging criteria is based on the following considerations:

Preliminary Round (submission criteria):

  • Submission of the matlab/simulink/python system model

  • Details descriptions of the report

    1.Provide important Labels in the schematic

    2.Describe the functionality, accuracy and design trade-offs

    3.Describe the implementation method of the neural network into analog circuit or FPGA.

Final Round (submission and Presentation criteria):

Analog Implementation

  • Video and Poster demonstration
  • Open access database of the schematic and layout

  • Details descriptions of the report

    1.Describe the implementation of the neural network to the circuit

    2.Describe the functionality of the circuit

    3.Describe the design trade-off of the circuit

    4.Provide Verification results including explanation. The verification includes schematic and post-layout functional verification and variability (includes PVT and Monte Carlo) analysis.

    5.Provide a detail circuit analysis using a mathematic model or with the use of circuit theory

Digital/FPGA implementation

  • Poster demonstration

  • Submission of the source code and project files (3 days before the final contest)

  • Details descriptions of the report

    1.Describe the implementation of the neural network to the circuit

    2.Describe the functionality of the circuit

    3.Describe the design trade-off of the circuit

[1] Hu, Y. and Loizou, P. (2007). “Subjective evaluation and comparison of speech enhancement algorithms,” Speech Communication, 49, 588-601. [pdf] (https://ecs.utdallas.edu/loizou/speech/noizeus)
